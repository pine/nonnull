## nonnull

:police_car: Simple JSR-305 Nonnull annotations

## Requirements
- Java 8 or later

## Getting started
The library is published to jcenter.

```gradle
repositories {
    jcenter()
}

depepdencies {
    implementation 'moe.pine:nonnull:0.1.0'
}
```

## Annotations
- NonNull
- NonNullApi
- NonNullFields
- Nullable
